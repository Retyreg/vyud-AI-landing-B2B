# Скриншоты Telegram Mini App — VYUD Frontline

Положи 4 скриншота в эту папку. Они подтянутся автоматически.

## Требования

| Файл | Экран | Рекомендуемый размер |
|---|---|---|
| `step-1.png` | Стартовый экран бота — список СОПов сотрудника | 400 × 800 px |
| `step-2.png` | Один из шагов обучения (текст + кнопка «Далее») | 400 × 800 px |
| `step-3.png` | Квиз — вопрос с 4 вариантами ответа | 400 × 800 px |
| `step-4.png` | Экран результата («Вы прошли / ваш балл») | 400 × 800 px |

## Формат

- PNG, прозрачный фон не нужен
- Соотношение сторон: **9:18** (портрет телефона)
- Размер файла: до 300 КБ на скриншот (сожми через TinyPNG если больше)

## Пока скриншотов нет

До добавления файлов в `phone-screen` показывается заглушка с иконкой и подписью.
Как только файл появится — заглушка исчезнет автоматически (`:has(img[src])` не нужен — img покрывает всю область).
